# ndi2025

Bienvenue sur le dépôt officiel de l'équipe IAMS3 de la **Nuit de l'info édition 2025** !

- Site web nuit de l'info : https://www.nuitdelinfo.com/
- Voir le site : (TO BE DEVELOPED)

La nuit de l'info est un concours de développement d'application web sur une nuit, de ~16h à ~8h. En plus d'un défi national imposé, chaque équipe peut choisir jusqu'à 5 défis à accomplir durant la nuit, chacun pouvant apporter des récompenses variées.

## Édition 2025
Cette édition 2025 était focalisée sur la démarche NIRD : un projet visant à promouvoir le **numérique libre, inclusif, responsable et durable** dans les **établissements scolaires**.

En plus de ce défi national, nous nous sommes engagés à accomplir les défis suivants :
- [**Rube Golberg**](https://www.nuitdelinfo.com/inscription/defis/496) où nous devions réaliser un élément du site de la manière la plus surcompliquée possible pour un résultat très minime.
- [**Ergonomie simplifiée**](https://www.nuitdelinfo.com/inscription/defis/444) qui nous demandait de développer un bouton sur notre site avec la pire experience utilisateur (UX) possible.
- [**Hair'gonomie**](https://www.nuitdelinfo.com/inscription/defis/479) qui, à l'inverse, récompense les équipes ayant accompli le défi principal avec la meilleure ergonomie.
- [**Hidden Snake**](https://www.nuitdelinfo.com/inscription/defis/483) nous invitant à cacher un jeu snake quelque part dans notre application web.
- [**Formulaire de la gloire**](https://www.nuitdelinfo.com/inscription/defis/491) demandant lui de créer un formulaire de contact le plus amusant et original possible.
